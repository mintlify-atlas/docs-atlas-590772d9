After running `threshold_RD()`, rrobot provides S3 `plot()` and `summary()` methods for every result class. These functions let you quickly inspect robust distances, threshold placements, and imputation results without writing custom visualization code.

## Plotting with `plot.RD()`

`plot.RD()` dispatches on an object of class `"RD"` (the return value of `threshold_RD()`) and supports three plot types controlled by the `type` argument:

| `type` | What it produces |
|---|---|
| `"histogram"` (default) | Histogram of squared robust distances on a log scale with a dashed threshold line. Observations are colored by their MCD inclusion status (included = green, excluded = orange). For `method = "all"` results, all method thresholds are overlaid as colored vertical lines. |
| `"imputations"` | Time-series plots per variable showing original data (black), temporal imputation (green dashed), all MI draws (blue, semi-transparent), one randomly highlighted MI draw (orange), and flagged outlier points (red). Only available when the result was produced with `method` set to `"MI"` or `"MI_boot"`. |
| `"univOut"` | Heatmap of the univariate outlier detection matrix — each cell represents one observation × variable combination, colored by whether it was flagged as a univariate outlier before the multivariate step. |

### Function signature

```r
plot(x, type = c("histogram", "imputations", "univOut"), method = NULL, ...)
```

When `type = "histogram"` and `method` is `NULL`, the method is auto-detected from the class of `x$thresholds`. You can override it by passing `method = "SI"`, `"SI_boot"`, `"MI"`, `"MI_boot"`, `"F"`, or `"SHASH"`. When `x$thresholds` is a list (i.e. the result was run with `method = "all"`), all non-SHASH thresholds are shown as overlaid colored lines in a single plot.

## Summarizing results with `summary()`

Each thresholding method has its own `summary()` method that prints key diagnostics to the console. Call `summary()` directly on the threshold result object (the `$thresholds` element of the `"RD"` object).

```r
# Setup used in all examples below
library(rrobot)
set.seed(42)
x <- matrix(rnorm(200 * 5), nrow = 200)
x[c(5, 42, 99), ] <- x[c(5, 42, 99), ] + 8
RD_obj <- compute_RD(x, mode = "auto", dist = TRUE)
```

### SI — Single Imputation

```r
result_si <- threshold_RD(x, RD_obj = RD_obj, method = "SI")
summary(result_si$thresholds)
```

```
Single Imputation (SI) Results
==============================
SI Threshold: 14.2371
Number of observations: 200
Robust subset size: 148
Outliers detected: 3 (1.5%)
```

Reports the threshold value, total observation count, number of observations in the MCD-included subset (`h`), and the count and percentage of observations flagged as outliers.

### SI_boot — Single Imputation Bootstrap

```r
result_siboot <- threshold_RD(x, RD_obj = RD_obj, method = "SI_boot", B = 1000)
summary(result_siboot$thresholds)
```

```
Single Imputation Bootstrap (SI_boot) Results
=============================================
Bootstrap samples: 1000
Confidence Interval:
  Lower bound (threshold): 13.8902
  Upper bound: 15.6117
```

Reports the number of bootstrap samples, the lower CI bound (used as the threshold), and the upper CI bound. The width of this interval reflects how much the threshold varies across bootstrap resamples.

### MI — Multiple Imputation

```r
result_mi <- threshold_RD(x, RD_obj = RD_obj, method = "MI", M = 50)
summary(result_mi$thresholds)
```

```
Multiple Imputation (MI) Results
================================
Number of imputations: 50
Threshold summary:
  Min: 12.4501
  Mean: 13.9872
  Max: 15.2340
Final threshold: 12.8013
```

Reports the number of imputed datasets, the distribution of per-imputation threshold values (min, mean, max), and the final threshold (lower CI bound across the `M` per-imputation quantiles).

### MI_boot — Multiple Imputation Bootstrap

```r
# With M=50, B=1000: total bootstrap samples = M * B = 50000
result_miboot <- threshold_RD(x, RD_obj = RD_obj, method = "MI_boot", M = 50, B = 1000)
summary(result_miboot$thresholds)
```

```
Multiple Imputation Bootstrap (MI_boot) Results
===============================================
Total bootstrap samples: 50000
Final threshold: 13.1245
Flagged outliers: 3 (1.5%)
```

Reports the total number of bootstrap samples (`M × B`), the final threshold, and the count and percentage of flagged outliers.

### F — Hardin-Rocke F-distribution

```r
result_f <- threshold_RD(x, RD_obj = RD_obj, method = "F")
summary(result_f$thresholds)
```

```
Hardin & Rocke F-Distribution Results
====================================
F-distribution threshold: 18.4762
Degrees of freedom: 5, 145.23
Consistency factor (c): 1.0614
Scale factor: 0.0283
Estimated m parameter: 150.23
```

Reports the threshold on the original (unscaled) robust distance scale, the F-distribution degrees of freedom (`df1 = p`, `df2 = m - p + 1`), the MCD consistency correction factor `c`, the scale factor used when comparing distances to the F quantile, and the estimated `m` parameter from the Hardin-Rocke finite-sample correction.

## Complete workflow example

```r
library(rrobot)

set.seed(42)
x <- matrix(rnorm(200 * 5), nrow = 200)
x[c(5, 42, 99), ] <- x[c(5, 42, 99), ] + 8

# Step 1: compute robust distances
RD_obj <- compute_RD(x, mode = "auto", dist = TRUE)

# Step 2: run default method (SI_boot)
result <- threshold_RD(
  x      = x,
  method = "SI_boot",
  RD_obj = RD_obj,
  B      = 1000,
  alpha  = 0.01
)

# Step 3: summarize the threshold result
summary(result$thresholds)

# Step 4: visualize RD histogram with threshold
plot(result)

# Step 5: visualize univariate outlier heatmap
plot(result, type = "univOut")
```
