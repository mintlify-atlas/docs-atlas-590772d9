rrobot is an R package for robust outlier detection in multivariate data. Unlike classical methods that assume Gaussian distributions, rrobot uses techniques—including the SHASH (Sinh-Arcsinh) transformation and Minimum Covariance Determinant (MCD) estimation—that remain effective when your data is skewed, heavy-tailed, or otherwise non-normal.

## What rrobot does

rrobot provides a two-stage outlier detection pipeline:

1. **Univariate pre-screening** — Each variable is individually screened for outliers using a robust empirical rule or an isolation forest. Detected outliers are imputed before the multivariate step.
2. **Multivariate robust distance** — Squared Mahalanobis distances are computed from a robust center and covariance (via MCD), then compared against a data-driven threshold.

Six thresholding methods are available, ranging from simple quantile-based rules to bootstrap and multiple imputation approaches.

## Get started

<Steps>
  <Step title="Install rrobot">
    Install from CRAN using a single command in your R console.

    ```r
    install.packages("rrobot")
    ```
  </Step>
  <Step title="Load the package">
    ```r
    library(rrobot)
    ```
  </Step>
  <Step title="Compute robust distances">
    ```r
    data("hbk", package = "robustbase")
    X <- as.matrix(hbk[, 1:3])
    rd <- compute_RD(X)
    ```
  </Step>
  <Step title="Detect outliers">
    ```r
    result <- threshold_RD(X, method = "SI_boot", RD_obj = rd)
    result$thresholds$flagged_outliers
    ```
  </Step>
</Steps>
